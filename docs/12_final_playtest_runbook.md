# Final Playtest Runbook

## 目标

在发布前完成最后一轮人工试玩与记录，重点确认：

- 菜单 -> 战斗 -> 结算全流程无断点
- 手感、难度和反馈是否稳定
- 暂停、最高分和重新开始流程是否正常
- 音量、音色和提示强度是否主观可接受

## 建议试玩轮次

至少完成 3 局：

1. 正常推进，尽量尝试通关
2. 主动多次暂停 / 继续 / 重开 / 回主菜单
3. 故意承伤，验证残机、复活、失败结算与最高分更新

## 每局需要观察的点

- 主菜单显示的最高分和最近一局结果是否合理
- 玩家移动、射击和拾取强化是否顺手
- Sector 1 到 Sector 2 的难度抬升是否自然
- Boss 警报、出场、受击和击破反馈是否清楚
- 结算页的得分、历史最高分和按钮流程是否正确
- 暂停菜单是否不会卡住流程
- 音效是否过刺耳、过密或过轻

## 完成标准

满足以下条件后，可以认为项目已适合做正式展示：

- 至少完成 3 局人工试玩
- 没有阻断流程的严重问题
- 最高分、暂停、结算都已人工确认
- 只剩少量主观打磨项，而不是结构性缺口
