# Release Notes v1.1.0

## 版本定位

`v1.1.0` 是在 `v1.0.1` 基础上继续完成战斗打磨、Boss 收口与 Windows 导出验证后的正式发布版本。

这一版的重点不是扩展系统，而是把现有完整可玩的复刻项目进一步打磨成更稳、更清晰、更适合交付和展示的版本。

## 相比 v1.0.1 的关键提升

- 重新调整双 Sector 的敌人编队、压力曲线和 Boss 前节奏
- Boss 增加低血量强化压迫与更明确的残血提示
- 敌人和 Boss 的命中反馈更有层次，战斗读感更清楚
- 继续优化敌我子弹的可读性
- 补齐 Windows 导出预设与分发说明
- 完成脱离编辑器的 Windows 构建导出和实际运行验证

## 本版亮点

- 完整主菜单 -> 战斗 -> 结算闭环
- 双 Sector 关卡结构
- 4 类基础敌人
- 3 类局内强化
- 2 个差异化 Boss
- 暂停流程、最高分记录、结算回路
- 更清楚的命中反馈、Boss 压迫感和战斗节奏
- Windows 导出链路已实际跑通

## 推荐 GitHub Release 标题

`Space War v1.1.0 - Combat Polish and Windows Build`

## 推荐 GitHub Release 简介

Space War v1.1.0 is the first release that combines post-release combat polish with a verified standalone Windows build.

This update focuses on enemy formation pacing, boss pressure tuning, hit feedback readability, and delivery readiness rather than expanding the game's scope. The result is a cleaner and more presentation-ready remake that stays faithful to the short-session arcade spirit of the original Nokia 3310 Space Impact.

## 建议附上的要点列表

- Refined two-sector pacing and pressure curve
- Stronger boss low-health behavior and warning feedback
- Clearer hit sparks and bullet readability
- Full arcade loop with score persistence and restart flow
- Verified standalone Windows export
- Apache-2.0 licensed repository
