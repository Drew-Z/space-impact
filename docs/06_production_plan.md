# 分阶段开发计划

## 已完成阶段回顾

### 阶段 0：预制作与文档

- 已完成项目骨架、参考拆解、核心循环、技术设计与测试计划初稿
- 已明确复刻边界：不做现代弹幕化，不做复杂成长，不做外层重系统

### 阶段 1：最小可玩原型

- 已完成菜单 -> 单关 -> Boss -> 结算的最小可玩闭环
- 已验证玩家移动、卷轴、射击、敌人、死亡、重开与回菜单

### 阶段 2：战斗与关卡扩展

- 已从单关扩展到五关 + 最终 Boss 的短局街机结构
- 已补充更多敌人编队、强化、Boss 差异和难度递进
- 已把武器成长扩展到 13 级，且保持局内简单成长

### 阶段 3：UI、菜单、结算与外层流程

- 已补齐暂停、HUD、主菜单摘要、最高分、结果页与操作提示
- 已让第一次进入的玩家能从菜单顺畅走到一局结束

### 阶段 4：打磨、平衡、音画与发布准备

- 已完成一轮手感、音画、反馈与文档收口
- 已完成导出、发布文档和发布前检查

## 当前版本状态

当前版本已经是可展示、可继续维护的复刻项目，重点不再是补大框架，而是做小步增强：

- 五关与最终 Boss 的节奏持续校准
- 13 级武器成长与“先掉武器、后掉 HULL”的惩罚联动平衡
- 敌人 / Boss 差异继续拉开
- 反馈与稳定性继续收口

## 当前迭代重点

### 版本目标

- 保持原版气质不跑偏
- 在现有结构上继续增强“短局高压 + 可重复游玩”
- 每次只推进一个明确主题，不做大而散扩展

### 当前已落地重点

- 武器升级改为 1-13 级成长线
- 5 级后火力开始加粗强化，9 级后切入贯穿到屏幕右侧的直线主炮
- 掉落改为“击毁数保底 + 随机补偿 + 固定节点”
- 新增 `spinner`、`sentinel` 等敌机与差异射击
- 扩展为五关 + 最终 Boss 的完整街机流程

## 下一阶段建议顺序

1. 继续做人工试玩驱动的平衡微调，重点看 Sector 4-5 与最终 Boss 的压力是否过长
2. 继续打磨高阶主炮的视觉层级，让 9 级以上的贯穿主炮更有压制感
3. 若确认五关结构稳定，再考虑补少量新敌机或 Boss 小招，而不是继续横向铺系统

## 风险控制

- 任何扩展若开始削弱短局感，就应立即收手
- 任何新系统若改变核心气质，就应退回到“更清晰的敌人、波次、Boss 差异”这条主线
- 若实现与文档不一致，优先更新文档并记录原因
