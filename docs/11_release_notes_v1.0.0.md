# Release Notes v1.0.0

## 版本定位

`v1.0.0` 是本项目第一个“完整可玩、可展示、可继续维护”的公开展示版。

它对应的是一个已经完成阶段 0 到阶段 4 收口的 Godot 4.6.1 复刻项目，目标是尽量还原 Nokia 3310《Space Impact》初代的核心体验，而不是做成现代原创弹幕射击。

## 本版亮点

- 完整主菜单 -> 战斗 -> 结算闭环
- 双 Sector 关卡结构
- 4 类基础敌人和更明确的波次节奏
- 3 类局内强化
- 2 个行为差异明显的 Boss
- 暂停流程、最高分记录、结算回路
- 统一的单色 / 低彩 LCD 视觉方向
- 程序化音效和基础音乐氛围
- 从预制作到发布收口的完整文档链路

## 推荐发布文案

Space War v1.0.0 is the first complete showcase build of this Godot 4.6.1 remake project inspired by the original Nokia 3310 Space Impact.

This version includes a full arcade run with a menu, two sectors, four enemy archetypes, three in-run power-ups, two bosses, pause flow, result screens, score persistence, and a cohesive monochrome retro presentation.

The project is intentionally scoped around the original game's short-session arcade feel instead of expanding into a modern bullet-hell or progression-heavy shooter.
