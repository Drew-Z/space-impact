## 当前已知限制

- Headless 退出时仍会看到 `ObjectDB instances leaked at exit` 警告
- 当前环境读取系统证书库会报 `Failed to read the root certificate store`
- 这两项目前未表现为可复现的玩法故障，但仍建议在本地编辑器和导出包中各实跑一轮确认

## 建议的人工试玩脚本

1. 从主菜单开始一局，确认菜单焦点、文案和最近一局摘要正常
2. 在一整局中至少体验一次 `repair`、`shield`、`overdrive` 与武器升级
3. 主动受击数次，确认先掉武器、再扣 HULL、护盾吸收和死亡逻辑
4. 至少打到 Sector 5，观察新敌人、新 Boss 与最终 Boss 是否仍保持原作式清晰感
5. 在战斗中触发暂停、继续、重开、回主菜单各至少一次
6. 完成或失败一局后检查结果页与高分更新

## 验证方法

- 优先执行可运行验证，而不是只看脚本
- 每次扩展后至少跑一次场景启动验证和一次长帧验证
- 若自动化环境受限，必须写明阻塞原因并提供本地验证路径
