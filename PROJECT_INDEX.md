- 运行项目：`project.godot`
- 默认主入口：`res://scenes/ui/main_menu.tscn`

## 目录速览

- `docs/`
  设计、技术、测试、发布说明和 postmortem。
- `scenes/`
  主菜单、战斗、HUD、结果页和实体场景。
- `scripts/`
  战斗逻辑、数据定义、自动加载与 UI 脚本。
- `assets/`
  美术、音频和字体资源。
- `build/`
  本地导出构建结果。
- `release/`
  正式分发包与版本目录。
- `tests/`
  验证脚本和测试资源。
