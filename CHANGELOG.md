# Changelog

本文档记录适合公开展示的版本节点。

## v1.1.1

当前正式发布版本，完成了从“完整可玩原型”到“更适合展示和维护的发布版”的收口。

### 本版重点

- 扩展为 5 个常规 Sector 加最终 Boss 阶段
- 武器成长重做为 13 级曲线，并在高阶形态启用持续直线主炮
- 玩家伤害模型调整为单命 `HULL 3`，并采用“先掉武器等级、后掉船体”的规则
- 主菜单补齐 `开始游戏 / 阶段继续 / 设置 / 退出` 结构，并加入中英文切换
- 每关背景、Boss 差异、Boss 支援波次与最终 Boss 收尾演出完成一轮强化
- 主菜单音乐、战斗音乐和关键战斗音效完成发布前打磨
- Windows 导出版已完成实际运行验证并发布

### 发布后收口

- 将敌人参数、Boss 支援规则与文案映射拆到 `stage_catalog.gd`
- 将关卡时间表拆到 `stage_schedule.gd`
- 将掉落与 Boss 警报等运行时平衡规则拆到 `run_balance.gd`
- 新增 `skirmisher`、`lancer` 两类后段敌人，强化 Sector 4-5 与 Boss 支援差异
- 更新 `README.md`、分发说明与发布后维护文档，方便继续迭代

### 当前状态

- 代码、文档、Release 和导出版状态一致
- `main` 分支已同步到远程仓库
- 项目已经进入“发布后维护与小版本迭代”阶段

## v1.1.0

面向后续维护的打磨版本，重点是基于发布后试玩继续提升忠实度、可读性和分发准备，而不是扩系统。

### 当前已完成

- 重新校准双 Sector 的敌人节奏与 Boss 前压迫感
- 为 Boss 增加低血量强化压迫与更明确的残血提示
- 增加敌人 / Boss 命中火花层次
- 继续优化敌我子弹可读性
- 完成 Windows 导出预设、分发文档准备和实际导出验证

### 当前状态

- 代码和文档已收口
- 启动与窗口渲染验证通过
- 脱离编辑器的 Windows 构建已完成一次实际运行验证

### 建议用途

- 作为当前更完整的 Windows 发布版本
- 作为展示和维护并行兼顾的稳定基线
- 作为后续只做轻量修正和资源替换的版本起点

## v1.0.1

基于人工试玩反馈完成的收口修正版，推荐作为当前正式 Release 使用。

### 本版新增 / 修正

- 修正主菜单与结算页布局偏移问题
- 放宽玩家活动范围，并以 HUD 顶栏 / 底栏作为边界
- 将武器升级节奏调整为“击毁累计保底 + 随机补偿掉落”
- 强化敌我子弹的形状、亮度与识别度差异
- 修正暂停时敌机、Boss、子弹与背景仍继续移动的问题
- 优化 Boss 预警表现、暂停菜单焦点稳定性和基础混音舒适度
- 补充最终人工试玩执行手册与试玩记录更新

### 本版重点

- 这一版不是扩系统，而是基于真实试玩把“能玩”推进到“更适合正式展示”
- 核心玩法结构保持克制，仍然围绕 Nokia 3310《Space Impact》初代的短局街机感
- 当前 `main` 分支已经比 `v1.0.0` 标签更接近可正式发布状态

### 建议用途

- 作为 GitHub Release 正式版本
- 作为项目展示、作品集或演示视频录制基础版本
- 作为后续仅做小幅平衡和资源替换的稳定基线

## v1.0.0

首个完整可玩、可展示的发布版本。

### 本版包含

- 完整主菜单到战斗到结算的一局闭环
- 双 Sector 关卡结构
- 4 类基础敌人
- 3 类强化
- 2 个差异化 Boss
- 暂停菜单、Game Over / 通关结算
- 最高分与局数记录
- 单色 / 低彩 LCD 风格 UI 与战斗画面
- 程序化音乐与音效
- 完整开发文档、测试文档与项目复盘

### 本版重点

- 优先还原 Nokia 3310《Space Impact》初代的短局街机感
- 保持系统克制，没有扩展成现代弹幕或复杂成长结构
- 项目结构已经收口，适合继续维护与展示

### 已知剩余项

- 仍建议补充人工完整试玩记录
- 音频与手感还可继续做主观调优
- 若作为更正式发布版本，可继续补充定制美术和音频资源
