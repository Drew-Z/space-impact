# Changelog

本文档记录适合公开展示的版本节点。

## v1.0.0

首个完整可玩、可展示的发布版本。

### 本版包含

- 完整主菜单到战斗到结算的一局闭环
- 双 Sector 关卡结构
- 4 类基础敌人
- 3 类强化
- 2 个差异化 Boss
- 暂停菜单、Game Over / 通关结算
- 最高分与局数记录
- 单色 / 低彩 LCD 风格 UI 与战斗画面
- 程序化音乐与音效
- 完整开发文档、测试文档与项目复盘

### 本版重点

- 优先还原 Nokia 3310《Space Impact》初代的短局街机感
- 保持系统克制，没有扩展成现代弹幕或复杂成长结构
- 项目结构已经收口，适合继续维护与展示

### 已知剩余项

- 仍建议补充人工完整试玩记录
- 音频与手感还可继续做主观调优
- 若作为更正式发布版本，可继续补充定制美术和音频资源
